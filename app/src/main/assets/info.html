PhotoTweak

PhotoTweak is used to control lighting levels. Each channel represents one or more lights attached to the physical fader channel. Connection to device is automatic, network configuarion is specified in settings. A demo mode is available if physical device is not present and can be turned in settings

Controls

 Level  Channel output power from 0 - 100%
 Fade  Rate at which channel transitions between levels (0 = instant change)
